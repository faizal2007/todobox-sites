## Technology Stack

- **Framework:** Flask 2.3.2
- **Database:** SQLAlchemy with multi-database support
- **Authentication:** Flask-Login with Google OAuth
- **Security:** Bleach (XSS), Werkzeug (password hashing), Flask-WTF (CSRF)
- **Frontend:** Bootstrap 4, Jinja2 templates
- **Production Server:** Gunicorn
- **Python:** 3.10+
