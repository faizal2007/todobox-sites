TodoBox

A production-ready Flask task manager with wisdom quotes, authentication, and multi-database support

Manage your tasks efficiently with a powerful REST API, secure user authentication, daily wisdom quotes, and flexible storage options. Built with Flask and modern best practices.

Get Started View on GitHub

Key Features

Everything you need for efficient task management

Todo Management

Complete CRUD operations for todos with RESTful API endpoints. Create, read, update, and delete tasks with ease.

Wisdom Quotes

Get inspired daily with wisdom quotes from the Zen Quotes API. Start each day with motivation and insight.

User Authentication

Secure JWT-based authentication with protected routes. Keep your todos private and secure.

Modern UI

Clean and intuitive Bootstrap-powered interface. Responsive design that works on all devices.

Security

Password hashing with Werkzeug, JWT token authentication, and secure session management built-in.

Flexible Storage

Support for SQLite, PostgreSQL, and MySQL databases. Choose the storage solution that fits your needs.

Quick Start

Get up and running in minutes

Installation
1. Clone the repository
git clone https://github.com/faizal2007/todobox.git
cd todobox
2. Install dependencies
pip install -r requirements.txt
3. Run the application
python app.py

The application will start on http://localhost:5000

API Examples
Register a new user
POST /api/register
Content-Type: application/json

{
  "username": "johndoe",
  "email": "john@example.com",
  "password": "securepassword123"
}
Login
POST /api/login
Content-Type: application/json

{
  "username": "johndoe",
  "password": "securepassword123"
}
Create a todo
POST /api/todos
Authorization: Bearer <your-jwt-token>
Content-Type: application/json

{
  "title": "Complete project",
  "description": "Finish the TodoBox project",
  "completed": false
}
Get daily quote
GET /api/quote
Authorization: Bearer <your-jwt-token>
Get all todos
GET /api/todos
Authorization: Bearer <your-jwt-token>
Configuration

TodoBox supports multiple database backends. Configure via environment variables:

# SQLite (default)
DATABASE_URL=sqlite:///todobox.db

# PostgreSQL
DATABASE_URL=postgresql://user:pass@localhost/todobox

# MySQL
DATABASE_URL=mysql://user:pass@localhost/todobox

# JWT Secret Key
SECRET_KEY=your-secret-key-here
