TodoBox

A modern, Flask-based Todo application with user authentication

Tech Stack

Built with modern Python web development technologies:

Flask 2.3.2 - Web Framework SQLAlchemy - ORM Flask-Login - Authentication Bootstrap 4 - Frontend Python 3.10+ - Runtime

Quick Start

Get TodoBox up and running in just a few steps:

1. Install Dependencies
pip install -r requirements.txt
2. Configure Environment
cp .flaskenv.example .flaskenv
3. Initialize Database
flask db upgrade
4. Run the Application
flask run

The application will be available at http://localhost:5000

API Example

TodoBox provides a RESTful API for managing todos. Here's a quick example:

Get All Todos
curl -X GET http://localhost:5000/api/todo
Example Response
{ "todos": [ { "id": 1, "title": "Sample Todo", "completed": false, "created_at": "2024-01-15T10:30:00Z" } ] }
